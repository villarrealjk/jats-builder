# XML JATS Builder

Aplicación web en Flask para crear artículos científicos y exportarlos en XML JATS.

## Funciones

- Gestión de artículos
- Autores, secciones, figuras, tablas y referencias
- Exportación XML JATS
- Sistema de usuarios
- Superusuario administrador
- Validación interna antes de exportar

## Instalación local

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
